How to load an audience into an ad set (quick steps)

  • Ad set → Audience section
  • Custom Audiences → Include → search the GTW audience name → add
  • For LAL stacks: add all three (1% + 3% + 5%) in the Include box — Meta dedupes overlap
  • When a LAL is the include, turn Advantage+ Audience OFF (so the LAL is the actual target, not just a suggestion) and Detailed Targeting Expansion OFF
  • Exclude box → add GTW_Mil_AllCustomers_7mo (+ the HighValue 1% on ALC campaigns, to keep them from competing with Max Val)
  • Publish

Priority order (if doing it in stages)

  • Auto Max Val + Home Max Val → HighValue LAL (the rescue — biggest impact)
  • Auto ALC + Home ALC → vertical LAL stacks
  • Auto BAU + Home BAU → suppression exclude + warm layer ad set

Rollout note

Load targeting, then give it 48-72 hours before judging — new audiences go through Meta's learning phase. Don't change budgets at the same time as targeting (we want to isolate what each change does).
