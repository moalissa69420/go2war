How to add an audience as a suggestion (steps)

  • Ad set → Audience section → keep Advantage+ Audience ON
  • In "Audience suggestions / Include these custom audiences" → search the GTW audience name → add it
  • In the Exclude box → add GTW_Mil_AllCustomers_7mo
  • Leave everything else default (Meta expands from the suggestion)
  • Publish

Priority order

  • Auto Max Val + Auto ALC → HighValue / Auto suggestions (biggest impact — rescue + revive)
  • Home Max Val + Home ALC → HighValue / Home suggestions
  • Auto BAU + Home BAU → confirm suppression exclude + add Warm Layer ad set

🚦 The most important rule — judge on lead QUALITY, not CPL

Advantage+ tends to lower CPL but can bring more unqualified leads. QuoteWizard pays on accepted leads. So track accept rate / conversion value, not just cost per lead. A cheap CPL with a low accept rate loses money. The Max Val (value-optimized) campaigns are our guardrail here.

⏳ After loading: leave it alone 3-5 days

New audiences enter Meta's learning phase (~50 conversions to stabilize). Don't change targeting or budgets mid-learning — it resets the algorithm. Load it, then let it run before judging.
